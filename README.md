# pickside-privacy
psikolojik ve sosyal konulardaki görüşlerinizi anonim olarak paylaşabileceğiniz bir anket uygulamasıdır. Kartları sağa, sola veya yukarı kaydırarak fikrinizi belirtin ve topluluğun dağılımını anında görün. Eğilimlerinizi keşfedin, en çok oylanan konuları inceleyin ve kendi profil istatistiklerinizi takip edin.
